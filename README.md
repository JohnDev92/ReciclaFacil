# ReciclaFacil

## Descrição
ReciclaFacil promove a gestão inteligente de resíduos para cidades sustentáveis. Atualmente contém imagens, arquivo index e configurações.

## Instruções para começar
1. Abra o arquivo `index.html` no seu navegador.
2. Siga as instruções na página para utilizar as funcionalidades do projeto.
